# Screenshot Placeholders

This directory contains screenshots for the README and documentation.

To generate screenshots:
1. Run the game: `python main.py`
2. Press F12 (if screenshot functionality added) or use external tools
3. Save screenshots with these filenames:
   - `gameplay.png` - Main combat action
   - `menu.png` - Main menu
   - `combat.png` - Intense combat with multiple enemies
   - `boss.png` - Boss battle scene
   - `upgrade.png` - Upgrade screen

Recommended resolution: 1280x720 or 1920x1080
Format: PNG with transparency where applicable
